1. Перейти в папку в GitBash, подключенную к Github 
   (с которой брали SSH-ключ)
2. Ввести команду 
   git clone https://github.com/vkinsu/NSU-OS-22932
3. Сделать любое изменение в клонированной папке 
   (создать любой файл)
4. Перейти в склонированную папку в GitBash (cd NSU-OS-22932)
5. Ввести "git add ."
6. git commit -m "любое сообщение на английском"
7. git push origin main
